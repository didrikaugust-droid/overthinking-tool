# 🧠 Overthinker Tool

En moderne nettside som hjelper deg å roe ned tanker.

## Funksjoner
- Flere typer svar (logisk, emosjonell, venn)
- Overthinking score
- Rask og enkel bruk
- Moderne design

## Hvordan bruke
1. Åpne index.html
2. Skriv hva du tenker på
3. Trykk "Analyser"
